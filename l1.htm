Red dead Redemption 2

Red Dead Redemption 2 is a 2018 action-adventure game developed and published by Rockstar Games. The game is the third entry in the Red Dead series and a prequel to the 2010 game Red Dead Redemption.

Minimum System Requirements-

OS: 64-bit Windows 10
Processor (AMD): Athlon X4 880K @4GHz or Equivalent
Processor (Intel): Core i3-6100 @3.7GHz or Equivalent
Memory: 8 GB
Graphics card (AMD): Radeon HD 7850 or Equivalent
Graphics card (NVIDIA): GeForce GTX 660 or Equivalent
Hard-drive space: 50GB

Installation Steps -

  1. Click on the download link.
  2. There will be 28 download links representing different parts of the download.
  3. Download and Extract them.
  4. Look for a file named Setup.exe
  5. Run the Setup.exe file as Administrator.
  6. Follow on screen instructions.
  7. After installation is completed, launch the game from the desktop shortcut.
  8. Enjoy!
